# ITニュース RSSリーダー

ブラウザだけで動作するシングルファイルのRSSリーダーです。サーバー不要、インストール不要で、HTMLファイルをダブルクリックするだけで使えます。

## 特徴

- **1ファイル完結** — `index.html` を開くだけで動作
- **サーバー不要** — ローカルファイルをブラウザで直接開ける
- **オフライン対応** — 取得済み記事はlocalStorageにキャッシュ
- **ダークテーマ** — 目に優しいダークデザイン
- **日本語UI** — 完全日本語対応

## 使い方

### 基本操作

1. `index.html` をブラウザで開く
2. 起動時に全フィードが自動取得される
3. 左サイドバーでフィードを選択して記事を絞り込む
4. 記事カードをクリックすると元記事が新しいタブで開く（自動既読）

### フィード管理

- **追加**: 右上の「フィード追加」ボタン → 名前とURLを入力して追加
- **削除**: フィード名の右の × ボタン
- **更新**: 右上の「全て更新」ボタン（または `Ctrl+R` / `Cmd+R`）

### 既読管理

- 記事を開く → 自動的に既読になる
- 各カードの「既読」ボタン → 手動で既読/未読を切り替え
- 右上の「全既読」ボタン → 表示中の全記事を既読にする
- 左サイドバーのフィルター → 未読/既読のみ表示

### 検索

上部の検索バーにキーワードを入力すると、タイトル・説明・フィード名でリアルタイムに絞り込めます。

## 技術仕様

| 項目 | 内容 |
|------|------|
| フロントエンド | HTML + CSS + JavaScript（バニラ） |
| スタイリング | Tailwind CSS（CDN経由） |
| RSS取得 | rss2json.com API（CORS回避） |
| データ保存 | localStorage（ブラウザ内のみ） |
| 外部依存 | Tailwind CSS CDN のみ |

### rss2json.com APIについて

CORSの問題を回避するために rss2json.com の無料APIを使用しています。

- 無料プランは1フィードあたり最大30件取得
- APIキー不要（無料プランの場合）
- レートリミット: 60リクエスト/時間（無料プラン）

## データの保存場所

すべてのデータはブラウザの `localStorage` に保存されます：

- `rss_feeds` — フィード一覧
- `rss_articles` — 取得した記事キャッシュ
- `rss_read` — 既読記事IDのリスト

ブラウザのキャッシュをクリアするとデータが削除されます。

## キーボードショートカット

| キー | 動作 |
|------|------|
| `Esc` | モーダルを閉じる |
| `Ctrl+R` / `Cmd+R` | 全フィード更新 |
